# Livox Mid360 Mapping and Navigation

A ROS-based mapping and navigation system for the Livox Mid360 LiDAR sensor.

## Overview

This project implements SLAM (Simultaneous Localization and Mapping) and autonomous navigation capabilities using the Livox Mid360 LiDAR sensor.

## Features

- Real-time 3D mapping with Livox Mid360
- SLAM integration
- Path planning and navigation
- Point cloud processing and visualization

## Hardware Requirements

- Livox Mid360 LiDAR sensor
- Compatible computing platform (recommended: NVIDIA Jetson or x86 with GPU)

## Software Requirements

- ROS (Robot Operating System)
- Ubuntu 20.04 / 22.04
- Livox SDK and ROS drivers
